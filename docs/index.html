حریم خصوصی و مدل‌ها

در حالت Local، inference روی گوشی انجام می‌شود. در حالت Cloud، context لازم برای انجام task به provider منتخب کاربر ارسال می‌شود. Accessibility و Notification Access قدرت بالایی دارند؛ فقط permissionهایی را فعال کنید که واقعاً لازم دارید.

  • عملیات ارسال پیام، تماس، auto-reply و ارسال فایل confirmation می‌خواهند.
  • مسیرهای deterministic برای کم‌کردن زمان و مصرف مدل، قبل از Agent اجرا می‌شوند.
  • آریا نباید دادهٔ دیده‌نشده را حدس بزند؛ خلاصه‌سازی chat/channel فقط از محتوای قابل‌مشاهده انجام می‌شود.

ساخت از سورس

cd arya-agent
./gradlew testDebugUnitTest
./gradlew assembleDebug
